Angular Material 主题系统（一）-- 使用介绍

发表于 2020-06-04 分类于 Angular
最近在自己的个人项目里用到了 Angular Material 组件库，也存在定制主题界面的需求，一番使用之后，这里参照官方文档，对它的用法做一个归纳和介绍。

Material 主题系统

在 Angular Material 中，主题是指颜色部分。按我的理解中，主题包含各类改变呈现效果，但不改变功能和内容的部分，因此这里也将文字排版纳入其中，包括使用方式和如何自定义。

主题颜色

包含一组 Angular Material 组件的颜色组合。按照 Material Design 的标准实现，每个颜色可以按（50，100，200…900）的层级，从浅到深调整。由下面这几种类型组成：

  • primary：主要颜色，广泛应用于应用是各个位置。
  • accent：突出色，用于 Material 设计特有的“FAB（浮动操作按钮）”，以及强调交互的元素的颜色。
  • warn：用于表示错误和警告状态。
  • foreground：用于文本和图标。
  • background：用于元素背景。

Angular Material 预先构建了几个内置主题文件。 这几个 css 文件除了包含主题颜色定义以外，还包含了所有 Material 组件的通用样式。因此，要使用 Material 组件库，只需在应用程序中引入一个单独的 CSS 文件即可。
预设文件从 @angular/material/prebuilt-themes/主题名称.css 引入，可用的预设主题有：

  • deeppurple-amber.css
  • indigo-pink.css
  • pink-bluegrey.css
  • purple-green.css

注意：如果应用程序内容未放置在 <mat-sidenav-container> 这个页面布局容器组件内，则需要将 .mat-app-background 类添加到外层元素（例如 <body>）中。 才能将主题背景应用于的应用。

文字排版

Angular material 的文字排版基于 material 设计规范，定义了多个层级，每个层级由 font-size，line-height 和 font-weight 三个样式配置组成。

定义的层级

层级 css 类 描述
display-1 ~ display-4 .mat-display-1 ~ .mat-display-4 巨大，一次性的标题性文字，常用于首页顶部
headline .mat-h1, .mat-headline 与 <h1> 标签对应的小节的头部
title .mat-h2, .mat-title 与 <h2> 标签对应的小节的头部
subheading-2 .mat-h3, .mat-subheading-2 与 <h3> 标签对应的小节的头部
subheading-1 .mat-h4, .mat-subheading-1 与 <h4> 标签对应的小节的头部
body-1 .mat-body, .mat-body-1 基本的正文文本
body-2 .mat-body-strong, .mat-body-2 较粗的正文文本
caption .mat-small, .mat-caption 较小的正文文本和提示文本
button 只用于组件中 按钮和链接
input 只用于组件中 表单输入字段

自定义主题

文字排版不是一成不变的，内置的颜色组合通常也不满足实际开发需要，并且在 Material 的图标之外，也时常需要加入自定义的图标，这就需要去自定义它们。

自定义 Material 组件颜色

要定义颜色组合很容易，因为 Material 的样式是用 Sass 开发的，所以要生成自定义的样式，其实就是传自定义的颜色变量，给 Material 的主题生成 mixin。

创建负责应用程序的整体主题样式文件：xxx-theme.scss，主要需要做四步：

  1. 导入 mat-core mixin。 这个 mixin 包含了 material 组件使用的所有通用样式。
    注意：只能引入一次，如果多次引入，引入几次，最后编译得到的 css 里就会包含几份相同的样式定义。

  2. 用多个颜色变量创建“调色盘”，这样就使得同一个颜色，有了多种色调的变化。

  3. 再用各个调色盘，组成主题对象。可以使用 mat-light-theme 或 mat-dark-theme 来创建明亮或者暗黑风格的主题对象。

  4. 最后，将此对象传递到 angular-material-theme，这个 mixin 将输出该主题的所有相应样式。

通常都是用 Angular-cli 创建项目，要使用自定义的这个主题，只需在 angular.json > styles 里，把这个 “xxx-theme.scss” 的路径引入。

使用文字排版工具

Angular Material 包含一组文字排版工具（mixin 和 function），可以用来自定义自己项目里的组件的文字样式，以保持和 Material 的样式定义一致。

  • mat-font-size($config, $level) - 通过配置对象和层级，获取 font-size。
  • mat-line-height($config, $level) - 通过配置对象和层级，获取 line-height。
  • mat-font-weight($config, $level) - 通过配置对象和层级，获取 font-weight。
  • mat-font-family($config) - 通过配置对象，获取 font-family。
  • mat-typography-level-to-styles($config, $level) - 一个方便生成样式的 mixin，通过传入配置对象和层级，快速生成一个 CSS 文字样式定义。
