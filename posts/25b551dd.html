Angular Material 的主题系统

发表于 2020-06-04 分类于 Angular

Material 主题系统

主题颜色

包含一组用于 Angular Material 组件的颜色，按照 Material Design 的标准实现。每个颜色可以按（50，100，200…900）的层级，从浅到深调整。每个主题颜色由下面这几个组成：

  • primary：主要颜色，广泛应用于应用是各个位置。
  • accent：突出色，用于 Material 设计特有的“FAB（浮动操作按钮）”，以及强调交互的元素的颜色。
  • warn：用于表示错误和警告状态。
  • foreground：用于文本和图标。
  • background：用于元素背景。

在 Angular Material 中，所有主题样式都在编译时静态生成，所以启动后不用花费运行周期来执行主题样式生成。

字体排版

预设主题颜色

Angular Material 预先构建了几个主题的 css 文件。 这些主题文件还包含核心的所有样式（所有组件共有的样式），因此只需在应用程序中为 Angular Material 库引入一个单独的 CSS 文件即可。
例如，可以从 @angular/material/prebuilt-themes 将主题文件直接包含到应用程序中，可用的预设主题有：

  • deeppurple-amber.css
  • indigo-pink.css
  • pink-bluegrey.css
  • purple-green.css

注意：如果应用程序内容未放置在 <mat-sidenav-container> 内，则需要将 .mat-app-background 类添加到外层元素（例如 <body>）中。 这样可以确保将主题背景应用于的页面。

自定义主题和字体

自定义主题

自定义主题需要做两个事：

  1. 导入 mat-core Sass mixin。 这个 mixin 包含了 material 组件使用的所有通用样式。 注意仅在应用程序中引入一次。 如果多次引入此 mixin，则最终得到的 css 将有这些通用样式的多个副本。

  2. 用多个调色板组成主题对象。 可以使用 mat-light-theme 函数或 mat-dark-theme 函数创建该对象。 然后，此函数的输出将传递到 angular-material-theme mixin，这个 mixin 将输出该主题的所有相应样式。
